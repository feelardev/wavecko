Jsme začínající Fivem Roleplay Server na Frameworku ESX-Legacy. Náš server byl původně založen 9.10.2021. Server byl v developingu cca. 3 měsíce. Každý den se na serveru pracovalo několik hodin za účelem udělat server co nejpříjemnější pro celou komunitu, což se do určité míry vydařilo, ale RP nebylo tak kvalitní jak jsme plánovali. Právě tohle byl jeden z hlavních důvodů proč jsme se rozhodli pro kompletní reborn serveru pod zbrusu novým názvem Arizona Roleplay. AZRP dopadl ještě hůř než původní NYRP, aktivita byla mnohem nižší a celý server obsahoval mnoho chyb. Je 1.10.2022 a Rozhodli jsme se udělat další nový server pod názvem The Rest Of Universe, kde chceme mít přátelskou komunitu, kvalitní RP, větší aktivitu. Jako admin team věříme že server dopadne přesně podle našich představ a stane se příjemným místem pro celou naší komunitu
